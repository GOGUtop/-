# ST-MoA-Collab 🤖
SillyTavern 多模型协作扩展（Mixture of Agents）

## 安装
1. 打开 SillyTavern → Extensions → Install Extension
2. 粘贴：`https://github.com/YourName/ST-MoA-Collab`
3. 启用即可

## 模式
- **串行精修**：A 写初稿 → B 修改 → C 再改…
- **并行汇总**：N 个模型各写一份 → 聚合器整合最佳输出

## 支持
任何 OpenAI 兼容 API（OpenAI / DeepSeek / Ollama / OneAPI / Claude 代理…）
